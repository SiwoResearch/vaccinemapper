# COVID-19 Vaccines Knowledge base

There are more than 200 vaccine candidates for COVID-19 in development. The Vaccine Mapper project is collating public information on COVID-19 vaccines aggregated from various sources and annotated to inform both the scientific community and the public on vaccine progress, provide new knowledge on the associations between vaccine components and immmunoloigical outcomes, enhance collaboration between vaccine developers, and help identify gaps in vaccine development.
